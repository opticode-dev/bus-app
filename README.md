## アプリ名
バスの時刻表検索アプリ

## 概要
大学間のバスの時刻表が検索できるWebアプリです。

## 使用技術
VSCode
Ubuntu
Linux
Python
Streamlit
GitHub

## 主な機能
カレンダーで日にち選択
出発地点選択
時刻表一覧表示

## 工夫した点
元々大学のHPに出てるバスの情報が複数あり、使いにくいと感じました。また、バスを利用するたびにPDFファイルを読み込まなくてはならず、面倒だと感じました。そこで、電車の時刻を検索できるアプリから着想を得て、バスの時刻表を検索できる簡易的なアプリを作りました。

## 今後の改善点
遅延情報などリアルタイム性の向上
